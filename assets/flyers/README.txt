Drop the flyers into this folder with these exact filenames:

  wine-down-mondays.jpg     (or .png)
  summer-soiree.jpg         (or .png)  -- the cocktails design
  live-music.jpg            (or .png)
  trivia-night.jpg          (or .png)
  voices-of-hawaii.jpg      (or .png)
  gentlemans-dinner.jpg     (or .png)
  girl-dinner.jpg           (or .png)

Each flyer is wired into its section by filename. Once a file lands
here, it appears immediately on the page. Until then, a striped
"flyer coming soon" placeholder shows where the flyer will go.

Note: summer-soiree.jpg currently holds the older "Live Jazz" band
design. To switch it to the new cocktails design, save that image
over assets/flyers/summer-soiree.jpg using the same filename.
